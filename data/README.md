# Data Directory

This repository does not include raw or processed data.

Download public data from the original providers:

- BTS Airline On-Time Performance: https://transtats.bts.gov/ONTIME/
- NOAA ASOS/METAR observations via Iowa Environmental Mesonet: https://mesonet.agron.iastate.edu/request/download.phtml
- OurAirports metadata: https://ourairports.com/data/

Use this structure:

```text
data/external/bts_on_time/
data/external/noaa_asos_metar/<year>/
data/processed/pressure_reliability/
```
